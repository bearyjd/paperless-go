Paperless Go is a mobile client for Paperless-ngx, the popular self-hosted document management system.

Manage your documents on the go with a clean, modern interface:
* Browse, search, and filter your document library
* View document details, thumbnails, and PDF previews
* Upload new documents via camera scan or file picker
* Receive shared files from other apps for quick upload
* Manage tags, correspondents, document types, and storage paths
* Edit document metadata and custom fields
* Create and manage custom field definitions
* Dashboard with library statistics overview
* Saved views: create, apply, rename, and delete filter sets
* Workflows: view, toggle, and delete automation rules
* PDF tools: rotate, split, compress, password protect, annotate, and redact
* Bulk operations: tag, re-tag, delete, merge, and more
* Biometric authentication for secure access
* Offline-capable with full SQLite caching
* Full-text search with autocomplete

Paperless Go connects to your self-hosted Paperless-ngx server using token-based authentication. You need a running Paperless-ngx instance to use this app.

Note: The F-Droid build does not include Google ML Kit (proprietary). Document deskew uses a pure-Dart algorithm and OCR-based metadata suggestions are disabled.
